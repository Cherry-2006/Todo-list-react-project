
# Todo-list Using React

Created a Todo list using React




## Installation
First Clone the project 
```bash
git clone https://github.com/Cherry-2006/Todo-list-react-project/
```
then run

```bash
npm i
```
to install all necessary modules to work

and finally
```bash
npm run dev
```
